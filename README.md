# "Choose Me" Game Concept

## 1. Overall Game Vision

This is a life simulator with RPG and gacha mechanics elements, inspired by the "Choose Me" web comic. Players will manage a world populated by unique, procedurally generated characters. The key feature is deep interaction of characters with the world and each other, powered by AI, as well as visual styling generated by neural networks.

**Genre:** Life Simulator, RPG, Sandbox.

**Key Features:**

- **Unique Characters:** Each character is randomly generated based on a set of tags (JSON) using AI, creating infinite variety of heroes.
- **Dynamic World:** Characters live their own lives, interact with the environment and other NPCs without direct player control.
- **Visual Flexibility:** Ability to switch between different visual styles (anime, realism, cinematic, cartoon) thanks to AI image generation.
- **Freedom of Choice:** Players can choose an era for their world: from fantasy and medieval to cyberpunk and steampunk.

**Main Functions:**
- RESTful API for client
- Authentication system (JWT)

- Game logic and procedural generation
